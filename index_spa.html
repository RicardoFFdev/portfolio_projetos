    Proyectos

    Proyecto de Pronóstico de Ventas: Rossmann Sales Forecast (Regresión de Aprendizaje Automático)

    Este proyecto tiene como objetivo estimar los ingresos de las tiendas de la cadena Rossmann. Realizaré una predicción de las ventas de las tiendas para las próximas seis semanas, con el objetivo de identificar qué tiendas tendrían suficientes ingresos para realizar nuevas inversiones. Se realizaron pruebas y entrenamientos de regresión con cinco algoritmos de Aprendizaje Automático, entre ellos, Random Forest Regressor, XGBoost Regressor y Linear Regression de Scikit-Learn. El XGBoost fue elegido como el modelo final de producción, su buen rendimiento de precisión y la rápida velocidad de ejecución fueron puntos diferenciales para este proyecto.
    Kaggle.

    Proyecto de Clasificación: "Hotel Chain Cancellation Rating"

    En esta competencia organizada por la Comunidad DS, el desafío propuesto es crear una solución de clasificación que brinde la mejor predicción posible para una cancelación de reserva realizada en una cadena hotelera. De esta manera, la cadena hotelera puede tomar las mejores medidas posibles, basadas en los perfiles de sus clientes, para evitar la cancelación de reservas ya realizadas.
    En esta competencia, los estudiantes de la Comunidad DS se dividieron en equipos de 5 estudiantes cada uno. En total, tuvimos 31 equipos y más de 180 participantes en total. Se realizaron pruebas y entrenamientos de clasificación con varios algoritmos de Aprendizaje Automático, como Random Forest Classifier, XGBoost Classifier y K-Nearest Neighbors.

    Proyecto de Análisis de Datos: Northwest Real Estate Agency

    En este proyecto de negocio, realicé el análisis de una base de datos real para la empresa ficticia "Northwest Real Estate Agency".
    La empresa "Northwest Real Estate Agency" compra y vende propiedades en la ciudad de Seattle (EE. UU.) y, para ello, la empresa requiere un análisis preciso de toda la información contenida en la base de datos.
    La solución final señala las mejores propiedades disponibles para la compra y, a continuación, calcula el precio de venta ya incluyendo el margen de beneficio.
