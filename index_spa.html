    Projeto de Análise de Dados: Northwest Real Estate Agency

    Neste projeto de negócio eu realizei a análise de um banco de dados real para a empresa fictícia "Northwest Real Estate Agency".
    A empresa "Northwest Real Estate Agency" compra e vende imóveis na cidade de Seattle (EUA) e, para isso, a empresa necessita de uma análise precisa de todas as informações contidas no banco de dados.
    A solução final aponta os melhores imóveis disponíveis para compra e, em seguida, calcula o preço da venda já acrescido da margem de lucro.
    * Este banco de dados público está disponível no website de competições de Ciência de Dados, o Kaggle.

    As Ferramentas utilizadas foram:

    • Python: Pandas, Numpy, Seaborn, Folium, Plotly e Plotly Express.
    • Anaconda, Pycharm Community, Visual Studio Code e Jupyter Notebook.

    Projeto de Previsão de Vendas: Rossmann Sales Forecast (Machine Learning Regression)

    Este projeto tem como objetivo estimar o faturamento das lojas da rede Rossmann. Farei uma previsão das vendas das lojas para as próximas seis semanas, com o objetivo de identificar quais lojas teriam faturamento suficiente para realizar novos investimentos. Foram realizados testes e treinamentos de Regressão com cinco algoritmos de Machine Learning, dentre eles, Random Forest Regressor, XGBoost Regressor e Linear Regression da Scikit-Learn. O XGBoost foi escolhido para o modelo final de produção, o seu bom desempenho de acurácia e a rápida velocidade de execução foram pontos diferenciais para este projeto. * Este banco de dados público está disponível no website de competições Kaggle./p>

    As Ferramentas utilizadas foram:

    • Python: Pandas, Numpy, Seaborn, Linear Regression, Random Forest Regressor, XGBoost Regressor, BorutaPy e Pickle.
    • Anaconda, Pycharm Community, Visual Studio Code e Jupyter Notebook.
    • Visualizações com Seaborn.
    • Cloud Computing com Render e Streamlit via Flask app.
    • Estatística Cross-validation.

    Projeto de Classificação: "Hotel Chain Cancellation Rating"

    Nesta competição realizada pela Comunidade DS o desafio proposto é a criação de uma solução de classificação que aponte a melhor previsão possível para um cancelamento de uma reserva realizada na rede hoteleira. Dessa maneira a rede hoteleira poderá tomar as melhores medidas cabíveis, embasada nos perfis dos seus clientes, para poder evitar o cancelamento de reservas já realizadas.
    Nesta competição os alunos da Comunidade DS foram divididos em Squad's com 5 estudantes cada. Ao todo tivemos 31 Squad's e mais de 180 participantes ao todo. Foram realizados testes e treinamentos de Classificação com vários algoritmos de Machine Learning, dentre eles, Random Forest Classifier, XGBoost Classifier e K-Nearest Neighbors.
    * Este banco de dados público está disponível no website de competições Kaggle.

    As Ferramentas utilizadas foram:

    • Python: Pandas, Numpy, Seaborn, Folium, Plotly e Plotly Express.
