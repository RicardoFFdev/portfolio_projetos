    Projects

    Sales Forecast Project: Rossmann Sales Forecast (Machine Learning Regression)

    This project aims to estimate the sales revenue of the Rossmann stores. I will make a sales forecast for the next six weeks, with the goal of identifying which stores would have enough revenue to make new investments.
    Regression tests and training were performed using five Machine Learning algorithms, including Random Forest Regressor, XGBoost Regressor, and Linear Regression from Scikit-Learn.
    XGBoost was chosen for the final production model due to its good accuracy performance and fast execution speed, which were differentiating factors for this project.
    * This public database is available on the Data Science competition website, Kaggle.

    The tools used were:

    • Python: Pandas, Numpy, Seaborn, Linear Regression, Random Forest Regressor, XGBoost Regressor, BorutaPy, and Pickle.
    • Anaconda, Pycharm Community, Visual Studio Code, and Jupyter Notebook.
    • Visualizations with Seaborn.
    • Cloud Computing with Render and Streamlit via Flask app.
    • Statistical Cross-validation.

    Classification Project: "Hotel Chain Cancellation Rating"

    In this competition held by the DS Community, the challenge is to create a classification solution that provides the best possible prediction for a cancellation of a reservation made in the hotel chain. This way, the hotel chain can take the appropriate measures, based on the profiles of its customers, to avoid cancellations of already made reservations.
    In this competition, the students from the DS Community were divided into squads of 5 students each. In total, we had 31 squads and over 180 participants.
    Classification tests and training were performed using various Machine Learning algorithms, including Random Forest Classifier, XGBoost Classifier, and K-Nearest Neighbors.
    * This public database is available on the Data Science competition website, Kaggle.

    The tools used were:

    • Python: Pandas, Numpy, Seaborn, Folium, Plotly, and Plotly Express.
    • Anaconda, Pycharm Community, Visual Studio Code, and Jupyter Notebook.
    • Interactive maps with Plotly and Folium.
    • Streamlit Cloud
    • Streamlit Python web framework.

    Data Analysis Project: Northwest Real Estate Agency

    In this business project, I performed the analysis of a real database for the fictional company "Northwest Real Estate Agency".
    The "Northwest Real Estate Agency" buys and sells properties in the city of Seattle (USA), and for that, the company needs an accurate analysis of all the information contained in the database.
    The final solution identifies the best properties available for purchase and calculates the sale price with the added profit margin.
