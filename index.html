    Projetos

    Projeto de Previsão de Vendas: Rossmann Sales Forecast (Machine Learning Regression)

    Este projeto tem como objetivo estimar o faturamento das lojas da rede Rossmann. Farei uma previsão das vendas das lojas para as próximas seis semanas, com o objetivo de identificar quais lojas teriam faturamento suficiente para realizar novos investimentos. Foram realizados testes e treinamentos de Regressão com cinco algoritmos de Machine Learning, dentre eles, Random Forest Regressor, XGBoost Regressor e Linear Regression da Scikit-Learn. O XGBoost foi escolhido para o modelo final de produção, o seu bom desempenho de acurácia e a rápida velocidade de execução foram pontos diferenciais para este projeto.

    Projeto de Classificação: "Hotel Chain Cancellation Rating"

    Nesta competição realizada pela Comunidade DS o desafio proposto é a criação de uma solução de classificação que aponte a melhor previsão possível para um cancelamento de uma reserva realizada na rede hoteleira. Dessa maneira a rede hoteleira poderá tomar as melhores medidas cabíveis, embasada nos perfis dos seus clientes, para poder evitar o cancelamento de reservas já realizadas.
