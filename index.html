    Projetos em Ciência de Dados

    Northwest Real Estate Agency

    Nesse projeto de negócio realizei a naálise de um banco de dados real, para a empresas fictícia Northwest Real Estate Agency. Este banco de dados público, foi disponibilizado pelo website de competições de Data Science, o Kaggle. A empresa Northwest Real Estate Agency tem como objetivo realizar a compra e venda de imóveis na cidade de Seattle. Para isso a empresa necessita de uma análise precisa de todas as informações e assim poder obter o lucro desejado na compra e venda de imóveis..

    As Ferramentas utilizadas foram:

    • Python, Pandas, Numpy, Seaborn, Folium, Plotly e Plotly Express.
    • Anaconda, Pycharm Community, Visual Studio Code e Jupyter Notebook.
    • Mapas interativos com Plotly e Folium.
    • Streamlit Cloud
    • Streamlit Python framework web.
